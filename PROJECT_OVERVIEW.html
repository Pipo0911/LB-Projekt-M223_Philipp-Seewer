✨ Funktionalitäten

Authentifizierung & Autorisierung

Registrierung
Neue Benutzer können sich registrieren. Passwörter werden mit BCrypt gehashed.
Login
JWT Token wird nach erfolgreicher Authentifizierung zurückgegeben (24h Gültigkeit).
ROLE_USER
Kann Spiele nur anzeigen (GET-Anfragen).
ROLE_ADMIN
Hat Zugriff auf alle CRUD-Operationen (Create, Read, Update, Delete).

Game Management

GET /api/games
Alle Spiele mit optionalen Filtern (Titel, installiert, Spielzeit, etc.)
GET /api/games/{id}
Einzelnes Spiel nach ID abrufen.
POST /api/games
Neues Spiel erstellen (nur ADMIN).
PUT /api/games/{id}
Spiel aktualisieren (nur ADMIN).
DELETE /api/games/{id}
Spiel löschen (nur ADMIN).
