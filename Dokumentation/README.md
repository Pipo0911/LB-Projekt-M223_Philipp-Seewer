# Steam Library – Projektdokumentation (M294)

**Modul:** M294 – Front-End Applikationen  
**Autor:** Philipp Seewer  
**Projekt:** Steam Library

Diese Dokumentation beschreibt die Entwicklung einer Web-Applikation
zur Verwaltung einer persönlichen Steam-Spielbibliothek.
Spiele werden über eine REST-API importiert, angezeigt und im Frontend bearbeitet.

Die Applikation wurde im Rahmen des Moduls M294 entwickelt
und erfüllt die geforderten technischen und funktionalen Anforderungen.
